An easy install/update script for nodejs removing the need for repos.

Download or git clone NodejsMe.sh and make executable.

Run with:
	     sudo ./nodejsME.sh

Available commands:
			install
			update
			version
			uninstall
			exit

Program will make a dot file in the same directory as itself named .njsme
the dot file in vital for prgram to run correctly so dont delete it.

nodejsME is intended for debian based linux distros only.
